# Assignment 2 - Control Structures in Python

This repository contains solutions for checking if a Number is Even or Odd and sum of Integers from 1 to 50 Using a Loop.

## Files
- `Assignment2_task1_even_odd.py` : Program to take a number and check if the Number is Even or Odd.
- `Assignment2_task2_sum_integers_1_50.py` : Find the sum of Integers from 1 to 50 Using a Loop.

## How to run
1. Make sure you have Python 3 installed.
2. From a terminal, run:
   - `python Assignment2_task1_even_odd.py`
   - `python Assignment2_task2_sum_integers_1_50.py`

## Output
Task 1:
Enter a number: 8
8 is an even number
Enter a number: 9
9 is an odd number

Task 2:
Sum of numbers from 1 to 50 is: 1275
